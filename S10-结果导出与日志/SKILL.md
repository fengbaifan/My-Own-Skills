# S10 结果导出与日志

## 定位

清洗流水线**收尾技能**。每轮清洗结束后，负责将判定结果转化为可交付的输出文件，并更新项目文档。

## 前置依赖

- S07-分批语义判定 当前轮次已完成
- S04-映射表构建 映射表已更新

## 输出文件规范

### 每轮输出

每轮清洗完成后，在新目录中生成两个文件：

```
0X_噪音去除第X次/
  ├── 噪音去除第X次_保留YYYY篇.[原格式扩展名]
  └── 噪音去除第X次_剔除清单ZZZZ篇.csv
```

#### 保留文件

- 从原始数据中提取 verdict ≠ NOISE 的记录
- **保持原始格式不变**（WoS纯文本保持WoS格式，CSV保持CSV格式）
- 记录顺序与原始数据一致
- 文件头/文件尾（如 WoS 的 FN/VR 和 EF）保留

#### 剔除清单 CSV

| 字段 | 说明 |
|------|------|
| seq | 映射表编号 |
| source_id | 来源平台唯一标识 |
| TI | 标题 |
| noise_type | 噪音类型编码 |
| reason | 判定理由（简要） |
| round | 判定轮次 |

- 编码：UTF-8 with BOM
- 仅包含本轮被判为 NOISE 的记录

### 最终输出

全部清洗轮次完成后，生成最终数据集：

```
NN_去噪完成_最终数据集/
  ├── 去噪完成_最终保留YYYY篇.[原格式扩展名]
  └── 去噪完成_剔除清单ZZZZ篇.csv
```

#### 最终保留文件

- 从**原始数据**中，按最终映射表的 seq 集合，一次性提取所有最终保留文献
- 不得从中间轮次的保留文件提取（防止字段完整性丢失）
- 确保与原始数据的字段完整性一致

#### 最终剔除清单

- 包含**所有轮次**被剔除的文献
- 包含各轮次的噪音类型字段（noise_type, noise_type_r2, noise_type_r3...）
- 可追溯每篇被剔除文献是在哪一轮、以什么理由被剔除的

#### 历史字段回写规范

最终剔除清单和映射表须包含以下历史追溯字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `verdict_history` | 字符串 | 判定历史，如 `R1:NOISE→R4:RELEVANT_CORE`，逗号分隔 |
| `noise_type_r[N]` | 字符串 | 第N轮追加的噪音类型（第N轮的 verdict=NOISE 时填写） |
| `last_verdict` | 字符串 | 最终判定结果（即当前 verdict） |
| `first_round` | 整数 | 首次被判定为 NOISE 的轮次 |

> 这些字段在每轮 S10 导出时更新下一轮的输入映射表。

## 数据校验

导出后执行严格校验：

| 校验项 | 公式 |
|--------|------|
| 记录数守恒 | 保留数 + 剔除数 = 输入总数 |
| 无遗漏 | 保留集中每条记录都能在原始数据中找到 |
| 无多余 | 保留集中没有被判为 NOISE 的记录 |
| 格式完整 | 输出文件格式正确可被目标工具加载 |

## 清洗日志更新

每轮导出后，更新 `00_清洗日志_修改原则与操作记录.md`：

### 日志模板

```markdown
### 第X轮清洗（YYYY-MM-DD）

**清洗对象**：`[输入文件路径]`（X,XXX 篇）

**清洗策略**：[本轮策略简述]

**噪音类型分布**：

| 编码 | 噪音类型 | 识别方法 | 剔除数 |
|------|----------|----------|--------|
| N[x] | [名称] | [方法简述] | XX |
| ... | | | |
| **合计** | | | **XXX** |

**清洗结果**：X,XXX − XXX = **X,XXX 篇保留**

**输出文件**：
- `0X_噪音去除第X次/噪音去除第X次_保留XXXX篇.[ext]`
- `0X_噪音去除第X次/噪音去除第X次_剔除清单XXX篇.csv`
```

### 日志文件清单维护

保持日志中的文件清单与实际工作区同步。

## 目录命名规范

```
01_原始数据/                       ← 严禁修改
02_文献分类映射/                   ← 映射表存放处
03_噪音去除第一次/                 ← 第一轮
04_噪音去除第二次/                 ← 第二轮
...
0N_噪音去除第N次/                  ← 第N轮
NN+1_去噪完成_最终数据集/          ← 最终输出
00_清洗日志_修改原则与操作记录.md   ← 操作日志
```

## 与其他技能的接口

- **输入来源**：S04（映射表）、S07（判定结果）
- **触发下游**：S11-批判性审查（每轮结束时可选触发）
