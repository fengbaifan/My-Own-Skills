# 文献计量分析核心算法与公式速查表

## 一、六大定律
1. **布拉德福定律 (Bradford's Law)**：核心期刊数量少但贡献大部分文献。
2. **帕累托定律 (Pareto Principle / 80/20)**：少数文献贡献多数引用。
3. **冯·海宁定律 (Lotka's Law)**：少数作者发表大量文献，多数作者发表少量文献。
4. **齐夫定律 (Zipf's Law)**：词频与排名成反比关系。
5. **普赖斯定律 (Price's Law / Sturgeon’s Law)**：少数核心文献或作者贡献大部分成果。
6. **布特定律 (Butcher's Law)**：科学产出集中性呈幂律分布。

## 二、引文计量指标
- **H指数**: H = max {h: 作者至少有 h 篇文献，每篇被引用 ≥ h 次}
- **G指数**: G = max {g: 前 g 篇文献总引用 ≥ g^2}
- **影响因子(IF)**: IF = 过去2年被引用次数 / 过去2年发表论文数

## 三、共现与网络分析
- **共引用**: w_ij = 被第三方文献同时引用 i 和 j 的次数
- **共词分析**: w_ij = 关键词 i 和 j 在同一文献中出现的次数
- **突现检测 (Burst Detection, Kleinberg)**: s(t) = k_t / λ_t
- **中心性指标**:
  - 度中心性 C_D(v) = 邻居数
  - 接近中心性 C_C(v) = 1 / Σ_u d(u,v)
  - 中介中心性 C_B(v) = Σ_{s≠v≠t} σ_st(v) / σ_st

## 四、主题与文本分析
- **TF-IDF**: TF-IDF_{t,d} = TF_{t,d} × log(N / DF_t)
- **余弦相似度**: sim(A,B) = Σ_i A_i B_i / (√Σ_i A_i^2 × √Σ_i B_i^2)
- **突现强度 (Burst Strength)**: s = log(p1 / p0) (状态跳变代价)
