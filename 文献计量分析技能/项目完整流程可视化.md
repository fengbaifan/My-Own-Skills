# 文献计量分析技能完整流程可视化

## 项目概览
该文档展示 S01–S16 技能的整体流程、输入输出及依赖关系，可作为团队快速理解和执行项目的可视化参考。

## 流程图（文字版）
```
S01-来源类型检测 → S02-来源数据分析 → S03-研究背景问询 → S04-映射表构建 → S05-检索策略解析 → S07-分批语义判定 → S10-结果导出 → S11-批判性审查 → S14-数据格式化导出 → S15-基础数据分析
                                                           │
                                                           └──→ S13-关键词突现分析（整合 S16-CiteSpace剪枝参考）
```

## 技能整合说明
- **S01-S05**：数据收集与清洗准备阶段
- **S06-S09**：数据处理、复核、噪音识别
- **S10-S11**：清洗结果导出与质量审查
- **S14-S15**：数据格式化与基础分析验证
- **S13-S16**：关键词突现与网络趋势分析，整合 CiteSpace 剪枝算法

## 输出文件及可视化
- 清洗后的文献数据 (WoS plain text)
- 基础统计 CSV 文件
- 关键词突现分析结果
- 网络可视化图（节点度、中心性、剪枝后的网络结构）
- 清洗日志文件

## 使用建议
- 按顺序执行 S01–S15 完成数据清洗与初步分析
- 使用 S13-S16 模块进行趋势与突现分析
- 所有技能遵循 SKILL.md 文档说明

> 可选：可将流程图制作为图形化图表 (如 Mermaid, PlantUML) 进一步可视化
