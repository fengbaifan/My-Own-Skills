# 主流程

```
S01-来源类型检测 → S02-来源数据分析 → S03-研究背景问询 → S04-映射表构建 → S05-检索策略解析 → S07-分批语义判定 → S10-结果导出 → S11-批判性审查 → S14-数据格式化导出
```

# 辅助流程

```
S06-超长文本读取（处理大文件）
S08-人工复核学习（策略优化）
S09-噪音模式库（模式沉淀）
S12-重复记录检测（可选去重）
S13-关键词突现分析（后处理趋势分析）
```

# 输入/输出规范

- 所有技能共享映射表字段:
  - seq, source_id, TI, SO, DE, ID, PY, VL, IS, BP, EP, DI, verdict, noise_type, reason, round
- 最终输出由 S14 生成 WoS plain text 文件，字段顺序与原始数据一致，直接可用 CiteSpace
