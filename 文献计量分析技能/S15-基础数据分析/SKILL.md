# S15 基础数据分析

## 定位

提供清洗后文献的初步分析能力，快速统计主要属性，帮助用户验证数据清洗结果的合理性。

## 输入

- S14-数据格式化导出的清洗完成文献文件（WoS plain text 格式）

## 输出

- 总文献数
- 作者总数
- 期刊总数及分布
- 年度文献分布
- 关键词统计
- 可选生成 CSV 或可视化图表

## 核心原则

1. 数据统计基于清洗后的保留文献
2. 保留原始字段完整性
3. 输出统计结果用于快速验证数据质量

## 操作流程

1. 读取 S14 输出文件
2. 按字段解析文献信息
3. 计算统计指标：文献总量、作者数量、期刊分布、年份分布、关键词统计
4. 可视化分布图（柱状图、折线图或饼图）
5. 输出 CSV 统计文件（可选）

## 与其他技能接口

- **前置**：S14-数据格式化导出
- **后续使用**：用户进行进一步分析或 S13-关键词突现分析
