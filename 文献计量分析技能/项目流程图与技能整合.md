# 文献计量分析技能流程图与整合概览

本文件展示 S01–S16 技能的流程关系、输入输出、算法与公式整合。

## 技能流程图（文字版）
```
S01 → S02 → S03 → S04 → S05 → S07 → S10 → S11 → S14 → S15
                                   │
                                   └──→ S13 (整合 S16) 
```

## 技能说明与接口

| 技能编号 | 技能名称 | 输入 | 输出 | 主要功能 | 参考算法/公式 |
|-----------|---------|------|------|-----------|---------------|
| S01 | 来源类型检测 | 原始文献 | 来源类型标记 | 识别数据来源平台 | - |
| S02 | 来源数据分析 | 原始文献 | 数据画像 | 字段完整性、文献类型、时间分布 | - |
| S03 | 研究背景问询 | 用户背景信息 | 研究边界参数 | 确定分析范围 | - |
| S04 | 映射表构建 | 原始数据 | 映射表 | 操作中枢 | - |
| S05 | 检索策略解析 | 映射表、检索式 | 噪音预期分析 | 检索策略风险建模 | - |
| S06 | 超长文本读取 | 大文件 | 分块文本 | 大文件迭代处理 | - |
| S07 | 分批语义判定 | 映射表、文本块 | 判定结果更新映射表 | 多维语义清洗 | - |
| S08 | 人工复核学习 | 人工复核结果 | 优化规则 | 学习复核策略 | - |
| S09 | 噪音模式库 | S07/S08 输出 | 模式库 | 噪音规则沉淀 | - |
| S10 | 结果导出与日志 | 映射表 | 保留/剔除文件、日志 | 清洗结果输出 | - |
| S11 | 批判性审查 | S10 导出结果 | 审查报告 | 六维审查 | - |
| S12 | 重复记录检测 | 映射表 | 去重结果 | 精确/疑似/版本去重 | - |
| S13 | 关键词突现分析 | S14 输出文件 | 突现分析结果、网络指标 | 趋势分析 | Kleinberg突现算法 + S16剪枝算法 |
| S14 | 数据格式化导出 | S10 输出文件 | WoS plain text | 格式化输出 | - |
| S15 | 基础数据分析 | S14 输出文件 | 文献统计、可视化 | 初步数据验证 | 计数统计、TF-IDF、余弦相似度 |
| S16 | CiteSpace剪枝参考 | 文献网络数据 | 剪枝方案、网络指标 | 提供剪枝算法参考 | Pathfinder、MST、Pruning Sliced Networks |
```

## 核心公式与算法参考

- H指数, G指数, 影响因子
- TF-IDF, 余弦相似度
- Kleinberg突现检测, 滑动窗口突现强度
- 中心性指标：度、接近中心性、中介中心性、聚类系数
- 剪枝算法：Pathfinder, MST, Pruning Sliced Networks
