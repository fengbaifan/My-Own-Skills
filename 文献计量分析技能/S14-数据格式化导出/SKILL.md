# S14 数据格式化导出

## 定位

清洗流水线**数据输出技能**。专门负责将 S10 导出的最终保留文献数据，格式化为原始 WoS plain text 格式，保持字段顺序和内容一致。

## 输入

- S10 导出的最终保留文件（保持字段完整）
- 映射表文件（包含 verdict、noise_type、reason 等字段）

## 输出

- 原始 WoS plain text 格式，字段顺序与原始数据一致
- 文件示例：`最终保留数据_WoS.wos`

## 核心原则

1. 保持字段完整性，字段顺序完全与原始 WoS 文件一致
2. 保留原始文献信息，不丢失任何字段
3. 字符编码统一为 UTF-8 / UTF-8 with BOM
4. 文件可直接导入 CiteSpace 使用

## 操作流程

1. 读取 S10 最终保留文件
2. 按原始 WoS 字段顺序重排字段
3. 输出到 WoS plain text 文件，文件名示例：`最终保留数据_WoS.wos`

## 与其他技能接口

- **前置**：S10-结果导出
- **辅助**：S06-超长文本读取（支持大文件分块）、S09-噪音模式库（可选保留附加标记字段）
- **后续使用**：CiteSpace 等文献分析工具
